Interactive Netflix Content Analytics Dashboard built using **Microsoft Power BI**, **Power Query**, and **DAX** to analyze titles, genres, ratings, release trends, and global content distribution.

# Project Overview

This project presents an interactive dashboard that provides insights into Netflix's content library through business intelligence and data visualization techniques.

The dashboard enables users to explore:

- Total Titles
- Movies vs TV Shows
- Rating Categories
- Genre Distribution
- Director Statistics
- Release Year Trends
- Global Content Distribution

The report follows a Netflix-inspired dark theme and focuses on delivering a clean, professional, and easy-to-understand user experience.

---

# Inspiration & Learning

This project was inspired by an existing Netflix Power BI dashboard that I came across during my Power BI learning journey.

Instead of simply recreating the original dashboard, I used it as a learning exercise and significantly redesigned multiple aspects of the report. I refined the overall layout, redesigned the KPI section, improved chart titles, enhanced the visual styling, updated the branding, improved the map visualization, simplified the user interface, and created a cleaner, more portfolio-ready dashboard.

This project reflects my practical understanding of Power BI dashboard design and my continuous effort to improve data visualization and storytelling skills.

# Dashboard Features

### KPI Overview
Provides a quick summary of important Netflix content metrics.

### Titles by Rating
Displays the distribution of Netflix titles across different maturity ratings.

### Content Type Distribution
Compares Movies and TV Shows using an interactive donut chart.

### Top Genres
Highlights the most common genres available on Netflix.

### Titles by Release Year
Visualizes content release trends over time.

### Titles by Country
Displays global distribution of Netflix content using a filled map.

### Interactive Filter
Allows users to filter the dashboard by content type (Movie or TV Show).
